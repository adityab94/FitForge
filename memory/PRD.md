# FitForge - Premium Fitness Tracking Dashboard

## Project Overview
FitForge is a comprehensive fitness tracking application pulled from GitHub repository: https://github.com/adityab94/FitForge

## Tech Stack
- **Frontend**: React 19, TailwindCSS, Radix UI components, Recharts
- **Backend**: FastAPI (Python), Motor (async MongoDB driver)
- **Database**: MongoDB with GridFS for file storage
- **Authentication**: JWT-based auth + Google OAuth

## Core Features
- User authentication (email/password + Google OAuth)
- Fitness data tracking and visualization
- Progress photos with GridFS storage
- Workout heatmaps
- Weight charts
- Body composition tracking
- Smart day logging
- Push notifications (VAPID)
